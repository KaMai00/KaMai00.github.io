Then, there was fire. Re-experience the critically acclaimed, genre-defining game that started it all. Beautifully remastered, return to Lordran in stunning high-definition detail running at 60fps. Dark Souls Remastered includes the main game plus the Artorias of the Abyss DLC.

Requirements:

MINIMUM

  1. Requires a 64-bit processor and operating system
  2. OS: Windows 7 64-bit, Service Pack 1
  3. Processor: Intel Core i5-2300 2.8 GHz / AMD FX-6300, 3.5 GHz
  4. Memory: 6 GB RAM
  5. Graphics: GeForce GTX 660, 2 GB / Radeon HD 7870, 2 GB
  6. Storage: 8 GB available space

  1. Requires a 64-bit processor and operating system
  2. OS: Windows 10 64-bit
  3. Processor: Intel Core i5-4570 3.2 GHz / AMD FX-8350 4.2 GHz
  4. Memory: 8 GB RAM
  5. Graphics: GeForce GTX 460, 1 GB / Radeon HD 6870, 1 GB
  6. Storage: 8 GB available space
